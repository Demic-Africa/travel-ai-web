# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in Demic Africa platforms, please email **security@demicafrica.com**

We ask that you:
- 🔒 Report privately, not publicly
- 📋 Provide detailed steps to reproduce
- ⏱️ Allow reasonable time for fixes before public disclosure

## What We Protect
- User authentication systems
- API keys and secrets
- Personal travel data
- Payment integrations

We appreciate responsible disclosure and will acknowledge your report within 48 hours.
